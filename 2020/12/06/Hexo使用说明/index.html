Hexo使用

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

生成

部署和测试

在本地启动一个服务,可以预览系统:

hexo s

清除缓存文件 (db.json) 和已生成的静态文件 (public)

hexo clean

发布:

hexo d 或者 hexo generate –deploy (可以简写为hexo g -d)
